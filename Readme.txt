Ellipse fitting by spatial averaging of random ensembles
------------------------------------------------------------

The contents of this code are provided without any warranty. 
They are intended for evaluational purposes only.

You may want to start by running the provided Demo_SpaMeCo_EllipseFit.m script 
for synthetic data or Demo_SpaMeCo_EllipseFit_RealData.m for real images.
